# Agent Rules

This directory contains agent behavior rules, coding standards, and review guidelines.

## Directory Structure

```
rules/
├── coding-standards/     # Language-specific coding standards
│   ├── java.md
│   ├── python.md
│   └── javascript.md
├── review-guidelines/   # Code review guidelines
│   ├── general-review.md
│   └── security-review.md
├── agent-behavior/      # Agent behavior and interaction rules
│   └── communication.md
└── README.md           # This file
```

## Rule Categories

### Coding Standards
Language-specific coding conventions, best practices, and style guides.

### Review Guidelines
Guidelines for conducting code reviews, including:
- Security review checklists
- Performance review criteria
- Architecture review principles
- Code quality standards

### Agent Behavior
Rules governing agent behavior and interactions:
- Communication guidelines
- Decision-making frameworks
- Error handling approaches
- Response formatting standards

## Adding New Rules

1. Create the appropriate subdirectory if it doesn't exist
2. Add rule files in Markdown format
3. Include clear descriptions and examples
4. Update this README with the new rule's description

## Usage

These rules can be referenced by skills or used independently to guide agent behavior.
