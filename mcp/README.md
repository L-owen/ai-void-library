# MCP Servers

This directory contains Model Context Protocol (MCP) servers for extending AI agent capabilities.

## Directory Structure

```
mcp/
├── server-name/          # Individual MCP server
│   ├── README.md        # Server documentation
│   ├── package.json     # Node.js dependencies (if applicable)
│   ├── requirements.txt # Python dependencies (if applicable)
│   └── src/            # Server source code
└── README.md           # This file
```

## What are MCP Servers?

MCP servers enable AI agents to:
- Interact with external APIs and services
- Access databases and data sources
- Execute tools and utilities
- Extend agent capabilities beyond built-in functions

## Adding a New Server

1. Create a new directory under `mcp/`
2. Add server implementation and documentation
3. Include setup instructions in the server's README.md
4. Update this README with a link to the new server
